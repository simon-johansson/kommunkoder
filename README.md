> All municipalities in Sweden with corresponding county and municipality codes (kommunkoder).


## Install

```sh
$ npm install --save kommunkoder
```


## Usage

```js
var kommunkoder = require('kommunkoder');

kommunkoder('Rainbow');
```

```sh
# creates a browser.js
$ npm run browser
```
